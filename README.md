# coolomina.dev
### First time cloning the repo
```bash
$ docker-compose build --no-cache
```

This command also applies when you've updated one of your gems.
```bash
$ docker-compose run web bundle install -j 10
```

### Development
```bash
$ docker-compose up
```
Then visit http://localhost:4000.
